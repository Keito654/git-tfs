## Agent skills

### Issue tracker

Issues live as local markdown files under `.scratch/<feature-slug>/`; no external tracker or PR-triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default role names used as-is (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
